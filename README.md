# Northwind-sales-eda-Task-5-
Exploratory Data Analyss(EDA) on the Northwind Sales dataset using Python, Pandas, Matplotlib and Seaborn.

## Project Overview
This project performs an Exploratory Data Analysis (EDA) on the Northwind Sales dataset using Python and Pandas in Jupyter Notebook. The aim is to explore the dataset, understand its structure, and generate insights through summary statistics and visualizations.

## Objectives
- Load and inspect the dataset.
- Check the dataset structure and data types.
- Identify missing values.
- Generate descriptive statistics.
- Analyze revenue across countries.
- Analyze revenue by product category.
- Visualize key trends using charts.

## Tools Used
- Python
- Jupyter Notebook
- Pandas
- Matplotlib
- Seaborn

## Dataset
The dataset contains sales records from the Northwind database, including:
- Order ID
- Order Date
- Customer ID
- Company Name
- Country
- City
- Quantity
- Product Name
- Category
- Unit Price
- Units in Stock
- Discount
- Revenue

## Exploratory Data Analysis
The following analyses were performed:
- Displayed the first few rows of the dataset.
- Checked dataset information using `df.info()`.
- Verified missing values using `df.isnull().sum()`.
- Examined dataset dimensions using `df.shape`.
- Generated summary statistics with `df.describe()`.
- Calculated total revenue.
- Analyzed product category distribution.
- Identified revenue by country.
- Identified revenue by category.
- Created charts to visualize findings.

## Key Insights
- Germany generated the highest revenue among all countries.
- Beverages recorded the highest revenue among all product categories.
- The dataset contains no missing values.
- Revenue varies significantly across different countries and categories.

## Conclusion
The analysis provides useful insights into sales performance and customer purchasing patterns. These findings can help businesses focus on high-performing markets and product categories to improve decision-making.
